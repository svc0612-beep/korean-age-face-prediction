\# korean-age-face-prediction



Korean face age prediction web project using a machine learning model.



\## Project Overview



This project predicts age from face images using a trained machine learning model and provides a simple web interface.



\## Main Files



\- `index.html` : main web page

\- `style.css` : web page styling

\- `app.js` : frontend logic

\- `server.py` : local web server

\- `model\_inference.py` : model prediction logic



\## Notice



Original AI Hub dataset files and uploaded user images are not included in this repository because of privacy, license, and file size issues.
